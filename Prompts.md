# Prompt Templates

1. Folder with Icon Prompt:

    "Color Name" color folder on a white background with an icon of an "Icon Name" on it.

2. Folder Only Prompt:

    "Color Name" color folder on a white background.

3. Icon Only Prompt:

    Icon of an "Icon Name".
